# 🎬 AI Shorts Studio

An open-source AI-powered YouTube Shorts automation platform.

## 🚀 Features

* 🤖 AI Story Generator (Ollama)
* 🎨 AI Image Generator (ComfyUI)
* 🎙 AI Voice Generator
* 🎬 FFmpeg Video Builder
* 📺 Automatic YouTube Upload
* 📈 SEO Title & Description Generator
* 🔄 Fully Automated n8n Workflows

---

## Tech Stack

* n8n
* Ollama
* ComfyUI
* FFmpeg
* PostgreSQL
* Redis
* Docker
